# Python API Gateway - Version 5

A complete Python API Gateway with AI-assisted development workflow.

## Quick Start

### Prerequisites
- Python 3.11+

### Install Dependencies
```bash
pip install -r requirements.txt
```

### Run the Gateway
```bash
uvicorn src.main:app --reload
```

### Install MCP Dependencies
```bash
cd mcp/gateway-mcp
pip install -r requirements.txt
```
