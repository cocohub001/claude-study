# Go API Gateway - Version 4

A complete Go API Gateway with AI-assisted development workflow.

## Quick Start

### Prerequisites
- Go 1.21+
- Python 3.10+ (for MCP server)

### Build and Run

```bash
# Build
go build -o gateway ./cmd/gateway

# Run
go run ./cmd/gateway
```

### Install MCP Dependencies

```bash
cd mcp/gateway-mcp
pip install -r requirements.txt
```
