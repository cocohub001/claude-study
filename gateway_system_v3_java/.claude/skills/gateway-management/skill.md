# Gateway Management Skill

## Description

This skill provides capabilities for managing the API Gateway, including route configuration, health monitoring, and traffic management.

## Triggers

This skill is activated when:
- User asks about gateway routes or configuration
- User wants to add, remove, or modify routes
- User needs to check gateway health or metrics
- User wants to manage backend services

## Capabilities

### Route Management
- Add new routes
- Remove existing routes
- Update route configurations
- List all routes

### Health Monitoring
- Check gateway health status
- View real-time metrics
- Analyze traffic patterns

### Configuration
- Update gateway settings
- Manage timeouts and limits
- Configure load balancing

## Files

- `skill.md` - This file (skill definition)

- `examples/` - Usage examples

## Integration

This skill integrates with:
- MCP Server for AI-assisted management
- Java Gateway Core for actual operations
- Monitoring systems for metrics
