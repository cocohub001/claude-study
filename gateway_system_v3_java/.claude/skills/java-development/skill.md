# Java Development Skill

## Description

This skill provides Java-specific development assistance for the gateway project, including code generation, best practices, and Spring Boot patterns.

## Triggers

This skill is activated when:
- User asks about Java code or patterns
- User needs help with Spring Boot configuration
- User wants to create new Java classes
- User needs help with Maven/Gradle

## Capabilities

### Code Generation
- Generate Java classes following best practices
- Create Spring Boot controllers and services
- Generate DTOs and entities

### Code Review
- Review Java code for issues
- Suggest improvements
- Check for common anti-patterns

### Spring Boot
- Configure Spring Boot applications
- Set up dependency injection
- Configure properties and profiles

## Best Practices

- Use constructor injection over field injection
- Follow SOLID principles
- Write unit tests for all business logic
- Use Optional instead of null checks
